Bienvenido a SteamInfo

Buscá cualquier perfil público de Steam ingresando el SteamID64 o la URL del perfil junto con tu API Key.

Cómo empezar

  • Obtené tu API Key gratis en steamcommunity.com/dev/apikey
  • Ingresá el SteamID64 (ej: 76561197960287930)
  • O pegá la URL completa del perfil de Steam
  • También funciona con URLs de vanidad (ej: steamcommunity.com/id/gaben)
  • Los perfiles privados mostrarán datos limitados
